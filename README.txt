clone this repository using this command -- git clone https://github.com/sharadvamsi/booking-activity.git

navigate to folder using command cd booking-activity.

rename sample.env file to  .env in root folder which is booking-activity and provide values for those keys.


run command "npm install" to install dependencies.
run command "npm start" to run the server.
now test apis


checkout demo video on how to test apis 

To test it localy use "http://localhost:5000" before every endpoint
To test it anywhere you can use vercel domain "https://booking-activity.vercel.app" before every endpoint
